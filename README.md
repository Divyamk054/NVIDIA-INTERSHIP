# NVIDIA AI & High-Performance Computing Summer Internship 2026

## Overview

This repository contains the work completed during the NVIDIA AI & High-Performance Computing Summer Internship Program 2026 conducted by the Presidency School of AI and Advanced Computing, Presidency University.

The internship provided hands-on exposure to Artificial Intelligence, Machine Learning, Deep Learning, Large Language Models (LLMs), Generative AI, GPU Computing, and High-Performance Computing using NVIDIA H200 Tensor Core GPUs.

---

## Program Details

* Duration: 2 Weeks (14 Days)
* Credit Hours: 2 Credits
* Total Contact Hours: 30 Hours
* Format: Lecture + Hands-on GPU Labs
* Hardware: NVIDIA H200 Tensor Core GPU
* Institution: Presidency University
